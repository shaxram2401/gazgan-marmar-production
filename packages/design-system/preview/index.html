05 — Foundation

Motion

Slow, cinematic easing. Hovers are confident; reveals are unhurried. Use cubic-bezier(.2,.7,.2,1) as the default.

Usage

Tokens are exported in three forms — use whichever fits the platform.

TypeScript
import { colors } from "@gazgan/design-system";
colors.gold.DEFAULT
Tailwind
extend: { colors: tokens.colors }
